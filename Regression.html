#Setting xic as the minimum value of x. 
xic = x.min()
x = np.array(df['side_a'])
#Sets the variable y to be a numpy array of the coloumn side_b in the dataset.
y = np.array(df['side_b'])
#Setting n to the amount of datapoints in each column in the pandas dataframe. This is 30 values, based on the dataset. 
n=df.shape[0]
'''Defining a function that takes 3 inputs(x,y,xic). This function will do calculations to return the slope and interception 
   of the datapoints that is fed to the function.'''
def regr_coeff(x,y,xic):
    #Calculation of xx, yy and xy.
    xx = x*x
    yy = y*y
    xy = x*y

    #To be able to determine the interception of the datapoints and the slope, Sxx, Syy and Sxy needs to be calculated. 
    Sxx = n*(xx.mean() -x.mean() * x.mean())
    Syy = n*(yy.mean() -y.mean() * y.mean())
    Sxy = n*(xy.mean() -x.mean() * y.mean())
    
    #From the values calculated above it is possible to determine the slope and intercept. These are then stored in the slope and icept variables.
    slope = Sxy/Sxx
    icept = np.mean(y) -slope*(np.mean(x) -xic)
    
    #The function returns the slope and icept variables as outputs. 
    return [slope, icept]

print(icept)

'''Defining a function named yline_from_coeff that calculates the regression line from the datapoints given as the 4 inputs(icept, 
   slope,xarray,xic). icept and slope was calulated in the function defined above as the regr_coeff function.'''
def yline_from_coeff(icept,slope,xarray,xic):
    #yline variable is assigned is the regressionline. This is calculated by icept+slope*(xarray,xic)
    yline = icept + slope*(xarray-xic)
    #The function returns the regressionline. 
    return yline

4. Describe the steps needed to compute bootstrap confidence intervals on a regression line.¶

To be able to compute bootstrap confidence intervals on a regression lines the following is needed: </br>

  1. Functions for creating a regression line and regression coefficient is needed. From the regression_coeff function slope and intercept is provided as outputs. From the regression line function, the regression line is created. </br>
  2. Further a for-loop needs to be defined. This goes from 0 to the amount of repretitions. Here bootstrap values from random integers created from the dataset needs to be collected. For every step in this loop the regression line and regr_coeff needs to be computed. </br>
  3. The values are saved in a new array that is the same size as the original data sample. Further using np.quantile the end values are computed from the new array. This is then used to plot the values that is situated in the interval set.

5. Explain the difference between the original sample size, the bootstrap sample size, and the number of bootstrap samples¶

Original sample size: This is the amount of datapoints that are in the original dataset. </br> The bootstrap sample size: The boostrap sample size is amount sampled from the original dataset. A dataset may contain few values, but when we bootstrap it we resample smaller samples many times. </br> Number of boostrap samples: The amount of times we sample the samller dataset and use the sample to display the properties of the original sample.

'''Defining a function that is used to calculate the correlation coefficient, this code is outlined above, 
   but now turned into a function. This could further be used to apply this several times if needed.'''
def compute_correlation_coefficient():
    #Calculation of xx, yy and xy.
    xx = x*x
    yy = y*y
    xy = x*y

    #To be able to determine the interception of the datapoints and the slope, Sxx, Syy and Sxy needs to be calculated. 
    Sxx = n*(xx.mean() -x.mean() * x.mean())
    Syy = n*(yy.mean() -y.mean() * y.mean())
    Sxy = n*(xy.mean() -x.mean() * y.mean())
    
    #Setting a variable named R to be equal to the equation that is needed to compute the correlation coefficient. 
    r = Sxy/np.sqrt(Sxx*Syy)
    return r 
#Printing the correlation coefficient.
