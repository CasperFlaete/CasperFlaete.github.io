#Defines a empty list.
difference = []
#Defines a variable that can be set to the difference. 
diff_number = 0;

#A loop looping trough the dataframe.
for x in df.itertuples(index=False):
    #Testing if Side A is smaller than Side B. 
    if x[2] < x[3]:
        #If Side A is smaller than Side B. Variable diff_number is sett to the difference. 
        diff_number = x[3]-x[2]
        #Adds the number to the list. 
        difference.append(diff_number)
    #Testing if Side B is smaller than Side A.
    if x[2] > x[3]:
        #If Side B is smaller than Side A. Variable diff_number is sett as the difference. 
        diff_number = x[2]-x[3]
        #Adds the number to the list.
        difference.append(diff_number)
    #Testing if Side A and Side B are equal. 
    if x[2] == x[3]:
        #If the sides are equal. The difference is zero, hence the viariable is set as zero. 
        diff_number = 0;
        #Adds the number to the list. 
        difference.append(diff_number)
#Prints the list. 
print (difference)

#Creating a n_boot variable and setting it to be 10^5
n_boot = 10**5

#Defining a variable that is set to be the difference with a dimension of n x n_boot. 
array = rnd.choice(difference, size = (n, n_boot))

#Defining boot_mean. Setting it to be the mean value of what has been calculated above. 
boot_mean = np.mean(array, axis = 0)


#Adding a line to represent the mean of the calculations done above. 
plt.axvline(array.mean(), color = 'red', linestyle = 'dashed')

#Adding lines to show 95%, 99% and 99,9% confidence intervals. 
plt.axvline(np.quantile(boot_mean, 0.95), color= 'grey', label='Confidence 95%')
plt.axvline(np.quantile(boot_mean, 0.05), color= 'grey')
plt.axvline(np.quantile(boot_mean, 0.99), color= 'black', label='Confidence 99%')

Discussion:¶

As can be seen in the plot above. The width is greatest for the mean that have the highest sample size. As it moves to 10^1 the width decreases. Since we are generating a bootstrap in this assignment the values will differ each time the code is ran. However, the general result of the approach is the same as it is run through.
